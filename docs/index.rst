assemblyinfo
==========

`Assemblyinfo <https://github.com/open2c/genomeinfo>`_ is a library to interact with genome metadata from different providers in python. This package is build on top of NCBI and UCSC data, allowing fast and accurate interconversions between chromosome names, latest assemblies and more!
